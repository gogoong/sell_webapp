# 外卖_webapp

> Vue、Vue-router、Vue-resource、better-scroll

## 安装步骤

``` bash
# 安装项目依赖
npm install

# 本地访问： localhost:8080
npm run dev

# 构建生产
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

>  prod_server.js

node prod_server.js

使用 express 启动应用服务.
