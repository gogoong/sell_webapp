# 外卖 Webapp

 > 相关技术栈

 Vue + Vue-router + Vue-resource + better-scroll + Less

>  安装步骤

``` bash

# 安装项目依赖
> npm install

# 本地访问： localhost:8080
> npm run dev

# 构建生产
> npm run build

# build for production and view the bundle analyzer report
> npm run build --report

```

> 项目所需数据

./data.json

>  生产环境下运行项目

``` bash
> node prod_server.js

```
